# Learn Spanish

An interactive terminal game that teaches Spanish from absolute beginner level. Built on [quest-engine](https://github.com/thorski1/quest-engine), guided by your warm and encouraging teacher **Sofia**.

## What You'll Learn

| Pack | Topics |
|------|--------|
| **Basics** | Alphabet, gender & articles, ser vs estar, conjugation, adjectives, question words, sentence building |
| **Greetings** | Hello/goodbye, introductions, how are you, please/thanks, formal vs informal, classroom phrases |
| **Numbers** | 1-100, big numbers, telling time, days & months, ordinal numbers |
| **Food** | Fruits & vegetables, drinks, meals, ordering at restaurants, cooking verbs, food culture |
| **Family** | Immediate & extended family, descriptions, house & rooms, pets, relationships |
| **Travel** | Transportation, directions, hotels, asking directions, airports, emergencies |
| **Daily Life** | Morning routines, school & work, hobbies, weather, shopping, body & health |
| **Culture** | Spanish-speaking countries, festivals, music & dance, famous people, traditions, regional differences |

Every challenge includes both **Spanish** and **English** so you build vocabulary and understanding from day one.

## Quick Start

```bash
# Install
pip install learn-spanish

# Play the full game
learn-spanish

# Play a single pack
basics-quest
greetings-es-quest
numbers-es-quest
food-es-quest

# Launch the web version
learn-spanish --web
```

## Requirements

- Python 3.10+
- A terminal with UTF-8 support (for accent marks and special characters)

## How It Works

Each skill pack contains themed **zones** with quiz challenges. Answer questions correctly to earn XP, level up, and unlock new zones. Sofia provides hints when you're stuck and teaches you something new after every answer.

The game uses spaced repetition principles -- concepts you struggle with appear more often, and each zone builds on what came before. Start with the Basics pack to build your foundation, then branch out into any topic that interests you.

## The 8 Skill Packs

1. **Basics** -- The essential building blocks: alphabet, grammar fundamentals, and sentence structure
2. **Greetings** -- Say hello, introduce yourself, and navigate polite conversation
3. **Numbers** -- Count, tell time, and use dates like a native speaker
4. **Food** -- Order at a restaurant, talk about cooking, and explore culinary culture
5. **Family** -- Describe your family, your home, and the people around you
6. **Travel** -- Get around, find your hotel, and handle emergencies abroad
7. **Daily Life** -- Talk about your routine, hobbies, weather, and health
8. **Culture** -- Explore the rich diversity of the Spanish-speaking world
